# 🍄 Projeto Mario & Luigi - Landing Page Responsiva

> 🌐 **Acesse o projeto online:** [Clique aqui para visualizar o site](https://lennercs.github.io/Land-page-Super-mario/)

Uma Landing Page de conversão única desenvolvida durante o bootcamp Fullstack, focada em capturar contatos de forma rápida e intuitiva. O projeto simula um site de serviços de encanamento dos irmãos Mario e Luigi, utilizando uma estética moderna e layout totalmente adaptável para qualquer dispositivo.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web fundamentais (Vanilla Tech), garantindo leveza e performance:

* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **CSS3:** Estilização visual avançada, layouts modernos com Flexbox e posicionamentos de camadas (`position`).
* **JavaScript:** Manipulação do DOM para interatividade suave e controle de fluxos de interface (como abertura da barra lateral).
* **Media Queries (@media):** Responsividade completa para garantir que o site fique perfeito desde telas pequenas (como iPhone SE) até monitores desktop.

---

## 📱 Funcionalidades & Aprendizados Técnicos

Este projeto foi excelente para consolidar conceitos essenciais de Front-End:

* **Responsividade Avançada:** Domínio do uso de `@media (max-width: 920px)` para reestruturar layouts horizontais de desktop em fluxos verticais fluidos no mobile (`flex-direction: column`).
* **Fluxo de Posicionamento:** Entendimento prático de como funcionam as propriedades `position: relative`, `absolute` e `fixed` (como o botão flutuante do WhatsApp), aplicando a lógica de camadas de forma limpa.
* **Espaçamento Estrutural vs. Deslocamento:** Diferenciação clara entre o uso de `margin` (que move o fluxo físico dos elementos) e propriedades de coordenadas como `top` ou `left`.
* **Manipulação do DOM:** Uso do JavaScript pontual e eficiente para gerenciar a experiência do usuário sem sobrecarregar a página.

## 💼 Potencial de Mercado (Modelo de Negócio)

Embora o tema seja focado nos personagens da Nintendo, a estrutura técnica deste projeto foi desenhada sob um modelo de **Landing Page de Conversão Única**, que possui altíssima saída no mercado de desenvolvimento freelancer. Ela é ideal para:

1. **Prestadores de Serviços Locais:** Encanadores, eletricistas, chaveiros e técnicos de manutenção que precisam de contato imediato via WhatsApp.
2. **Profissionais Autônomos de Alto Valor:** Advogados, nutricionistas ou personal trainers que utilizam o formulário lateral para captação de leads.
3. **Estética e Agendamentos:** Barbearias premium ou estúdios de tatuagem que usam vídeos de fundo para demonstrar a qualidade do serviço.
